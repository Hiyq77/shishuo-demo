# 06 AI需求

## 8. AI 需求（统一原则）

### 8.1 能力清单（MVP）

- 应用内模型对话窗口：**协作教研**、**课堂观察**均支持多轮自然语言对话  
- 智能识别与任务路由：支持自动识别「对话模式 / 文稿编辑 / 结构化生成」并选择处理方式  
- Agent 调度（MVP 仅 3 类）：**教案 Agent**、**课堂观察 Agent**、**通用对话 Agent**；导图/图谱/课件/练习等 Agent 见 `04-10-Agent调度.md` 后续版本说明  
- 协作教研：提纲、纪要、争议点/行动项；教案初稿、改写（多指令）；课堂笔记整理（与 `04-04` 对齐）  
- 课堂观察：简评、改进建议、结构化报告

### 8.2 质量与安全原则

- 输出需**结构化**（标题、列表），便于入库与展示。  
- 明确提示：AI 生成内容需教师审核后使用。  
- 支持「重新生成」与「人工编辑优先」。

### 8.3 模型与数据（PRD 层）

- 记录：调用类型、对象 ID、用户 ID、时间（用于排错与计费预留）。  
- 训练使用：默认 **不用于训练**（除非产品政策另有规定，需法务确认）。
