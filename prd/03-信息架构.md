# 03 信息架构

## 5. 信息架构（IA）

```
登录/注册（首期弱化，后补）
首页（模块化：顶栏 / 通知栏 / 应用广场 Icon / 右侧「师说」助手 / 页脚）
├── 协作教研（统一应用：教研主题、教案等，见 04-04）
├── 课堂笔记（独立应用，见 04-06）
└── 课堂观察
帮助与合规
```

说明：登录后默认进入 **首页**；**不再单独要求**「传统左右双栏工作台 + 独立应用广场页」的信息结构，应用入口以首页 **Icon 宫格** 为主（详见 `04-功能需求/04-00-首页.md`）。原「工作台」聚合能力（待办、粗统计等）并入首页 **通知栏** 等模块。  
**命名对齐**：PRD 中不使用「教研空间」「知识库」作为独立业务模块名；听评课/自评等能力统一以 **「课堂观察」** 作为应用与导航名称（见 `04-05-课堂观察.md`）；**「协作教研」涵盖原「教案生成」能力**（详见 `04-04-协作教研.md`）；**「课堂笔记」为独立应用**（详见 `04-06-课堂笔记.md`）。
