# 04-03-应用广场

### 6.3 应用广场（首页内嵌模块）

- 应用广场为 **首页主内容区内的模块化区块**，以 **Icon + 名称** 展示，**非**单独「应用广场列表页」必选（若保留独立 URL，仅作首页同构重定向或锚点亦可）。  
- **MVP 展示名称与路由**（与 `04-00-首页.md` 一致）：**协作教研**、**课堂笔记**、**课堂观察**（仅此三枚必选 Icon）。  
- 每个入口：**Icon、名称、一句话价值（可选）、上次使用时间（有则显示）**；点击进入 **该应用独立下钻页**（列表/首页）。  
- **扩展**：题库、教师成长记录等通过配置增加 Icon 占位，未开放时可灰显或「敬请期待」。

**验收**：在首页应用广场内，**协作教研**、**课堂笔记**与**课堂观察**三枚 Icon 可点击并进入对应应用首页/列表；扩展位展示符合配置与权限。


**输入/输出**：

- 输入：应用配置、用户最近使用时间。  
- 输出：应用卡片列表、进入动作、上次使用信息。

**状态流转**：

- 应用可用状态：可用 -> 维护中（占位）。  
- 用户触达状态：未使用 -> 已使用（更新最近使用时间）。

**边界情况**：

- 应用暂不可用时置灰并展示维护提示。  
- 上次使用时间为空时不展示该字段。

**数据埋点（MVP）**：

- `app_market_viewed`：进入应用广场（可与 `home_viewed` 合并统计，以首页曝光为准）。  
- `app_card_clicked`：点击应用卡片。  
- `app_entered`：成功进入目标应用。
