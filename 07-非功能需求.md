# 07 非功能需求

## 9. 非功能需求（MVP）


| 类别  | 要求                               |
| --- | -------------------------------- |
| 性能  | 列表页首屏 < 2s（弱网可降级）                |
| 可用性 | 核心流程 3 步内可理解；关键页有空状态引导           |
| 安全  | HTTPS；密码加密存储；基本权限校验              |
| 审计  | 创建人、更新时间（列表可见）                   |
| 兼容  | Web 端优先（Chrome/Edge/Safari 主流版本） |
