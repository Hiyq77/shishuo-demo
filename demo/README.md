# 师说 · PRD 前端 Demo（静态 MVP）

本 demo 严格对应 `prd/` 中 **首页**、**协作教研**、**课堂观察**（含上传必填字段）、**资源库** 的最小流程；**全站顶栏**展示平台名称、日期（`MM.DD.YYYY 星期X`）、用户信息（演示占位）。样式参考柔和圆角、浅色 pastel 卡片风。

## 如何运行

任选其一：

```bash
cd demo && python3 -m http.server 5173
```

浏览器打开 `http://127.0.0.1:5173/`。

或直接双击打开 `index.html`（部分浏览器对 `file://` 限制较少；若遇路由问题请用本地服务器）。

## 页面与 PRD 映射

| 路由 | PRD 文档 |
| --- | --- |
| `#/` | `prd/04-功能需求/04-00-首页.md` |
| `#/research` | `prd/04-功能需求/04-04-协作教研.md`（列表占位） |
| `#/classroom` | `prd/04-功能需求/04-05-课堂观察.md`（列表 + 上传 drawer） |
| `#/repo` | `prd/04-功能需求/04-10-资源库.md`（列表占位） |

顶栏用户信息为 **展示占位**（PRD：由需求方 API 提供，C 端不维护）。
